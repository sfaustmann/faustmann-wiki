# Vorbemerkung

Dieses Dokument listet die wichtigsten Dragon-Sprachbefehle auf, um Browser-Funktionen mit der Dragon Browser-Erweiterung per Sprache zu steuern. Es hilft bei der effizienten Navigation, Auswahl und Steuerung von Webinhalten in Microsoft Edge und Google Chrome. Die Befehle basieren auf den [offiziellen Dragon-Hilfeseiten](https://www.nuance.com/products/help/dragon161/dragon-for-pc/deu/dlg-vla/Content/GettingStarted/web_browsing_TOC.htm#Im_Web_browsen).

Voraussetzung für die Verwendung sind die korrekt installierten Browser-Add-Ons für Google Chrome und Microsoft Edge für die entsprechende Dragon-Edition. Eine umfassende Installationsanleitung dazu finden Sie in unserem PDF-Dokument „Übersicht der Browser Add-Ons und Installation der Erweiterungen für Dragon 2025“.

# Microsoft Edge – Sprachbefehle

In Microsoft Edge stehen primär Navigations- und Tab-Steuerungsbefehle zur Verfügung. Eine direkte sprachliche Interaktion mit Webseitenelementen ist eingeschränkt.

## Grundlagen

|  |  |
| --- | --- |
| **Aktion** | **Sprachbefehl** |
| Edge starten | Öffne Microsoft Edge |
| Liste von Web-Befehlen öffnen | Was kann ich sagen |

|  |  |
| --- | --- |
| **Aktion** | **Sprachbefehl** |
| 1. Neue Webseite öffnen | Drücke Alt D (Der Cursor wechselt in die Adressleiste.) |
| 2. Diktieren der Webadresse | a) Diktieren Sie die Webadresse ins Diktierfenster und klicken Sie auf **Übertragung**, oder sagen Sie „Klick Übertragung“. |
|  | b) Schließen Sie das Diktierfenster und diktieren Sie direkt in die Adressleiste. Wenden Sie diese Option an, wenn Sie die volle Textsteuerung nicht benötigen. |
| 3. Diktierte Webseite öffnen | Drück Eingabetaste |

## Tabulatoren und Navigation

|  |  |
| --- | --- |
| **Aktion** | **Sprachbefehl** |
| Zwischen Feldern wechseln | Tabulatortaste |
| Neuen Tab öffnen | Öffne Tab |
| Tab schließen | Tab schließen |
| Nächster Tab | Gehe / wechsele zum nächsten Tab |
| Vorheriger Tab | Gehe / wechsele zum vorherigen Tab |
| Erster Tab | Gehe / wechsele zum ersten Tab |
| Letzter Tab | Gehe / wechsele zum letzten Tab |
| Zwischen mehreren Tabs wechseln | Gehe zum Tab \<*zwei, drei usw.*\> |

|  |  |
| --- | --- |
| **Aktion** | **Sprachbefehl** |
| Dragon öffnet die vorherige Seite Ihrer Browser-Chronik (Zurück) | Geh zurück |
| Dragon öffnet die nächste Seite Ihrer Browser-Chronik (Vorwärts) | Geh vorwärts |
| Seite neu laden | Drücke F5 |
| Seite nach unten scrollen | Gehe *\<Zahl\>* Zeilen nach unten / Seite nach unten |
| Seite nach oben scrollen | Gehe *\<Zahl\>* Zeilen nach oben / Seite nach oben |
| Ans Seitenende scrollen | Drücke Steuerung Ende |
| An Seitenanfang scrollen | Drücke Steuerung Oben |

## Listenfelder und Webseiten-Steuerelemente

Sagen Sie „Klick“, gefolgt vom ganzen oder teilweisen Namen des Steuerelements. Wenn Sie in den Einstellungen (Dragon-Leiste \> Einstellungen \> Optionen \> Befehle) die Option **'Klick' sagen, um Schaltflächen und andere Steuerelemente auszuwählen** deaktiviert haben, müssen Sie nicht zusätzlich „Klick“ sagen, um ein Steuerelement anzuklicken. Sagen Sie dann einfach den Namen des Steuerelements. Mehr dazu auf der [Seite "Befehle" im Dialogfenster "Optionen"](https://www.nuance.com/products/help/dragon161/dragon-for-pc/deu/dlg-vla/Content/DialogBoxes/options/options_dialog_startshut_tab.htm).

### Listenfelder anklicken

|  |  |
| --- | --- |
| **Aktion** | **Sprachbefehl** |
| Listenfelder anklicken | Klick Listenfeld |
| Option aus der Liste auswählen | Klick \<Option\> |

### Andere Steuerelemente anklicken

|  |  |
| --- | --- |
| **Aktion** | **Sprachbefehl** |
| Link anklicken | Klick \<Linkname\> |
| Button anklicken | Klick \<Buttonname\> |
| Textfeld aktivieren | Klick \<Feldname\> |
| Kontrollkästchen/Checkbox auswählen | Klick \< Kontrollkästchenname\> |
| Optionsschaltfläche auswählen | Klick \<Optionsschaltflächenname\> |
| Listenfeld öffnen | Klick Listenfeld |
| Listenwert auswählen | Klick \<Listenwert\> |
| Bild anklicken | Klick \<Bildname\> / Klick Bild (gefolgt von „Nimm \<Zahl\>“) |
| Eingabefeld auswählen | Klick \<Eingabefeldname\> |

### Steuerelemente anklicken, wenn diese mehrfach vorkommen

1. Sagen Sie einen der folgenden Befehle:
- Klick Textlink
- Klick Schaltfläche
- Klick Kontrollkästchen
- Klick Editierfenster
- Klick Bild
- Klick Optionsschaltfläche
- Klick Eingabefeld
- Klick Text schreiben

*Es erscheinen neben allen Steuerelementen dieses Typs Zahlen.*

#### 2. Sagen Sie „Nimm \<Zahl\>“, um ein Steuerelement anzuklicken.
#### 3. Zahlen ausblenden:

Wenn Sie keine Zahl auswählen, sagen Sie einen der folgenden Befehle, um die Zahlen wieder auszublenden:

- „Abbrechen“
- „Rückgängig machen“

### Installation in Google Chrome

- Klicken Sie im Google Chrome-Browserfenster rechts oben auf das 3 Punkte Menü (…) und wählen den Menüpunkt „Erweiterungen“. Im weiteren Popup klicken Sie auf den Link „Chrome Web Store besuchen“.
- Im Chrome Web Store geben Sie links oben in die Suche „Dragon Weberweiterung“ ein. Als Suchergebnis wird die Dragon Weberweiterung angezeigt, klicken Sie dieses Suchergebnis an.
- Klicken Sie nun auf „Hinzufügen“.
- Bestätigen Sie die folgende Abfrage mit der Schaltfläche „Erweiterung hinzufügen“.
- Die Dragon-Weberweiterung wird im Google Chrome Browser installiert und Sie erhalten eine Bestätigung.
