Vorbemerkung

Dieses Dokument listet die wichtigsten Dragon-Sprachbefehle auf, um Browser-Funktionen mit der Dragon Browser-Erweiterung per Sprache zu steuern. Es hilft bei der effizienten Navigation, Auswahl und Steuerung von Webinhalten in Microsoft Edge und Google Chrome. Die Befehle basieren auf den offiziellen Dragon-Hilfeseiten.

Voraussetzung für die Verwendung sind die korrekt installierten Browser-Add-Ons für Google Chrome und Microsoft Edge für die entsprechende Dragon-Edition. Eine umfassende Installationsanleitung dazu finden Sie in unserem PDF-Dokument „Übersicht der Browser Add-Ons und Installation der Erweiterungen für Dragon 2025“.

Microsoft Edge – Sprachbefehle​

In Microsoft Edge stehen primär Navigations- und Tab-Steuerungsbefehle zur Verfügung. Eine direkte sprachliche Interaktion mit Webseitenelementen ist eingeschränkt.

Listenfelder und Webseiten-Steuerelemente​

Sagen Sie „Klick“, gefolgt vom ganzen oder teilweisen Namen des Steuerelements. Wenn Sie in den Einstellungen (Dragon-Leiste > Einstellungen > Optionen > Befehle) die Option 'Klick' sagen, um Schaltflächen und andere Steuerelemente auszuwählen deaktiviert haben, müssen Sie nicht zusätzlich „Klick“ sagen, um ein Steuerelement anzuklicken. Sagen Sie dann einfach den Namen des Steuerelements. Mehr dazu auf der Seite "Befehle" im Dialogfenster "Optionen".

Bild anklickenKlick <Bildname> / Klick Bild (gefolgt von „Nimm <Zahl>“)
Eingabefeld auswählenKlick <Eingabefeldname>

Steuerelemente anklicken, wenn diese mehrfach vorkommen​

  1. Sagen Sie einen der folgenden Befehle:
  • Klick Textlink
  • Klick Schaltfläche
  • Klick Kontrollkästchen
  • Klick Editierfenster
  • Klick Bild
  • Klick Optionsschaltfläche
  • Klick Eingabefeld
  • Klick Text schreiben

Es erscheinen neben allen Steuerelementen dieses Typs Zahlen.

2. Sagen Sie „Nimm <Zahl>“, um ein Steuerelement anzuklicken.​

3. Zahlen ausblenden:​

Wenn Sie keine Zahl auswählen, sagen Sie einen der folgenden Befehle, um die Zahlen wieder auszublenden:

  • „Abbrechen“
  • „Rückgängig machen“

Installation in Google Chrome​

  • Klicken Sie im Google Chrome-Browserfenster rechts oben auf das 3 Punkte Menü (…) und wählen den Menüpunkt „Erweiterungen“. Im weiteren Popup klicken Sie auf den Link „Chrome Web Store besuchen“.
  • Im Chrome Web Store geben Sie links oben in die Suche „Dragon Weberweiterung“ ein. Als Suchergebnis wird die Dragon Weberweiterung angezeigt, klicken Sie dieses Suchergebnis an.
  • Klicken Sie nun auf „Hinzufügen“.
  • Bestätigen Sie die folgende Abfrage mit der Schaltfläche „Erweiterung hinzufügen“.
  • Die Dragon-Weberweiterung wird im Google Chrome Browser installiert und Sie erhalten eine Bestätigung.
